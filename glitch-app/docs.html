Heater triggers

Power on of off your heater

Add the following paths to this resource path base: https://maker.ifttt.com/

Request Path Description
POST /trigger/HeaterOn/json/with/key/{your-auth-key} Powers on your heater
POST /trigger/HeaterOff/json/with/key/{your-auth-key} Powers off your heater

Light triggers

Power on of off your light, or change the light colour

Add the following paths to your socketxp URL.

Note: This project uses socketxp to expose Flask app APIs with secure public sockets. For more information, see SocketXP setup.

Request Path Description
GET appliances/light?power=status Returns the power status of your light: {"power":"on"} or {"power":"off"}
POST appliances/light?power=on Powers on your light
POST appliances/light?power=off Powers off your light
POST appliances/light?colour={colour} Change the colour of your light

Heater energy usage

Get energy usage information for your heater

Add the following paths to your socketxp URL.

Note: This project uses socketxp to expose Flask app APIs with secure public sockets. For more information, see SocketXP setup.

Request Path Description
GET appliances/energy/heater?power=status Returns the power status of your heater: {"power":"on"} or {"power":"off"}
POST appliances/energy/heater?power=on Starts the power usage meter for your heater
POST appliances/energy/heater?power=off Stops the power usage meter for your heater
